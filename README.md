## About The Project

![Screenshot 2024-07-04 210023](https://github.com/MohsenSangSefidi/Stadino/assets/160093991/d7f7492c-44bd-4d2a-a146-db96632a7032)

در این پروژه، ما یک وب سایت فروشگاهی کتاب کامل و کاربردی را با استفاده از فریمورک قدرتمند جنگو پیاده‌سازی کرده‌ایم. جنگو به عنوان یکی از محبوب‌ترین فریمورک‌های پایتون، امکان توسعه سریع و امن وب اپلیکیشن‌ها را فراهم می‌کند.


### Built With

* Django-Rest-Framework
